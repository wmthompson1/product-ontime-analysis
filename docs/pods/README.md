# Pods

Saved agent chat responses worth keeping. When the user says **"save to pods"**,
the agent writes the referenced response as a dated markdown file in this
folder so it is git-tracked, greppable, and easy to copy from the editor —
no cursor-painting in the chat pane.

Naming: `YYYY-MM-DD_short-topic.md`. Each file opens with the date and the
question/context it answered.
