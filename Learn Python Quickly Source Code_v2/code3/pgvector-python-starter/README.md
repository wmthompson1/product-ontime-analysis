## Setup

1. Install PostgreSQL 15 and pgvector
2. Run `init_db.sql`
3. Fill in `.env`
4. Run:
   - `python insert_vectors.py`
   - `python query_vectors.py`

## Dependencies

- sentence-transformers
- psycopg2
- pgvector extension for PostgreSQL
