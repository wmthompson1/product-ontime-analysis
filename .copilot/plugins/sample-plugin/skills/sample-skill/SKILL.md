# Sample Skill

## Purpose
Provide a short, practical workspace orientation for this repository.

## When to use
- User asks for a quick project orientation.
- User asks what to run first in this repo.
- User asks for immediate next steps after setup.

## Response format
1. Give a 3-bullet orientation summary.
2. List 3 practical next commands or actions.
3. Mention one common pitfall to avoid.

## Guardrails
- Keep output concise and actionable.
- Prefer repository-relative paths.
- Do not include secrets or credentials.
